MainStem plugin for MVP vendors who use WooCommerce.

== Description ==
This plugin allows communication between the MainStem MVP program and a WooCommerce install.

== Installation ==
Just install the MainStem WooCommerce plugin, set the API key that was given to you by MainStem and that\'s it!  Your WooCommerce products will automatically sync with the MainStem system.  When an order is created on MainStem, the order will instantly show up right in your WooCommerce.
